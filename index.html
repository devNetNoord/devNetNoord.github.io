Introductie: Distributed Application Runtime (Dapr)

Eelco Koster / 13:00 - 13:45 / Conferentiezaal II

Dapr is een, door Microsoft ontwikkeld, open-source framework dat een set aan bouwstenen voor microservices levert voor gedistribueerde applicaties. Die bouwstenen voor bijvoorbeeld pub/sub, state management, service discovery en triggers kunnen op iedere infrastructuur draaien en zijn programmeertaal onafhankelijk te gebruiken. In deze introductie sessie leg ik uit hoe Dapr werkt en hoe je het kunt toepassen om een schaalbare microservice architectuur te bouwen.

Serverless functions everywhere with OpenFaas

Erwin Staal / 14:00 - 14:45 / Auditorium

Serverless is a relatively new offering that is hard to miss nowadays and you’ve probably already heard a lot about it. OpenFaaS is a framework for building Serverless applications. It makes it really simple to turn anything into a serverless function that runs on Linux or Windows through Docker Swarm or Kubernetes. By using one of these orchestrators you can now run serverless functions on your own private cloud or in any public cloud such as Azure by using AKS. And since it’s based on Docker containers it doesn’t matter if you want to use Go, Java, Python, C#, Django, ASP.NET Core, or anything else. In this session I’ll introduce you to serverless and OpenFaas. We will see how simple it is to get OpenFaas up and running and deploy your first function. We will dive into a few ways to trigger your functions, look at authentication and secrets and see how we can monitor them using Open-source tooling.

No more secrets! Securing Azure cloud applications using managed identities

Tjeerd-Menno Douma / 14:00 - 14:45 / Conferentiezaal I

Het beheren van “secrets” als database connection strings, access keys en credentials kan in cloud applicaties een flinke uitdaging zijn. Zeker wanneer de cloud applicatie uit een groot aantal verschillende services bestaat. Deze “secrets” veilig houden is belangrijk want het lekken van deze “secrets” zou zo maar zeer ernstige gevolgen kunnen hebben. In deze sessie leer je hoe met Azure Managed Identity je cloud applicatie kunt beveiligen zonder gebruik te hoeven maken van “secrets”.

Process IoT data with Microsoft Orleans

Andre Broers / 14:00 - 14:45 / Conferentiezaal II

To manage the energy transition we need to process a lot of data in a scalable solution. In this presentation I will show how we use the actor pattern to aggregate energy data from domestic smartmeters. I will show the actor pattern principles and how to implement it in the Dotnet Core Microsoft Orleans framework.
