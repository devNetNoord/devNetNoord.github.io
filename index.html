Lessons learned: Implementing an event sourced, microservices architecture on Azure

Cornell Knulst en Pascal Naber / 19:00 - 20:00

One of the most used buzzword nowadays in IT is Microservices. But what are the challenges and pitfalls you face when implementing such an architecture? How to identify the right set of microservices? How to ensure for example a transactional scope between event processing and publishing? How about cross-cutting concerns like monitoring and logging?

In this session we will walk you through a real-world case of building a distributed application based on a microservices using .NET Core and Docker. During the session we point out lessons learned during the process of building this application.
